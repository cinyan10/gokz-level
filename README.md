# gokz-level

A SourceMod plugin that displays a player’s [gokz.top](https://gokz.top) rating using Faceit-style level icons on the scoreboard.

![image of the plugin](image.png)

## Commands

- `!rating` — shows your current skill score and level.

## Requirements

- `smjansson`
